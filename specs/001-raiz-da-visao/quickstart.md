# Quickstart Validation Guide: Raiz da Visão

## Prerequisites
- Node.js (v18+)
- Local development server ready to run the Next.js application

## Setup
1. From the repository root, install dependencies if not already done:
   ```bash
   npm install
   ```

## Run Validation
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open a web browser and navigate to the Raiz da Visão page (e.g., `http://localhost:3000/raizdavisao` - exact route to be determined during implementation).

## Expected Outcomes
- **SC-001 Validation**: The page should display the Raiz da Visão content:
  - "O que é o projeto" (Introduction)
  - Os Pilares (ÀLÀGBÁRA and ÀṢẸÀGBÀ)
  - Medicinas da Floresta (Rapé, Ayahuasca, Sananga)
  - O Propósito e Chamado
- **Constitution Check**: Visually verify that the page meets the Constitution principles:
  - The color palette is earthy and serene.
  - Typography is modern but respectful.
  - No overly playful micro-animations exist.
  - The tone is serious and reflects "Respeito à Tradição Viva".
